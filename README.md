## 🚀 ขึ้นออนไลน์ด้วย GitHub Pages (ฟรี)

### 1. สร้าง Repository
1. ไปที่ [github.com](https://github.com) → ล็อกอิน
2. กด **"New repository"** → ตั้งชื่อ เช่น `juice-delivery`
3. เลือก **Public** → กด **Create repository**

### 2. อัปโหลดไฟล์
1. คลิก **"uploading an existing file"**
2. แตกไฟล์ ZIP แล้วลากไฟล์ทั้ง **8 ไฟล์** นี้ขึ้น:
   ```
   index.html       ← ตัวแอปหลัก
   images.js        ← รูปสินค้าทั้งหมด
   sw.js            ← Service Worker (offline)
   manifest.json    ← ข้อมูล PWA
   icon-192.png     ← ไอคอนแอป
   icon-512.png     ← ไอคอนแอป (ใหญ่)
   README.md        ← ไฟล์นี้
   .gitignore
   ```
3. กด **"Commit changes"**

### 3. เปิด GitHub Pages
1. ไปที่ **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** → **/ (root)** → กด **Save**
4. รอ ~2 นาที → แอปพร้อมที่:  
   `https://<username>.github.io/juice-delivery/`

## 📱 ติดตั้งแอปบนมือถือ

เมื่อเปิดเว็บในมือถือ จะมีแถบ **"ติดตั้งเลย"** ขึ้นมาอัตโนมัติ กดได้เลย!

| ระบบ | วิธีติดตั้ง |
|------|------------|
| **Android** (Chrome) | แถบ "ติดตั้ง" → กดยืนยัน |
| **iPhone/iPad** (Safari) | ปุ่มแชร์ (↑) → "เพิ่มไปยังหน้าจอโฮม" |
